# 🎬 AI Movie Studio Creator (Text-Based)

An AI-powered web application that generates structured movie content including **plots, screenplay dialogue, and story ideas** using a locally hosted language model.

---

## 🚀 Problem Statement

Creating compelling movie stories, dialogues, and structured plots requires significant time, creativity, and effort. Many users struggle to transform raw ideas into well-structured narratives.

The objective of this project is to:

> Develop an AI-driven platform that assists users in generating complete movie-related textual content such as plots, dialogues, and story structures in an efficient and interactive manner.

---

## 💡 Solution

This system provides a unified interface where users can input basic details such as genre and theme, and receive structured, AI-generated movie content.

---

## 🧠 Features

* **Plot Generation**

  * Generates logline and structured story (three-act format)

* **Screenplay Dialogue Generation**

  * Produces cinematic-style dialogues

* **AI Chat Assistant**

  * Helps refine ideas and expand story concepts

* **Local AI Processing**

  * Runs entirely on a local model without external APIs
